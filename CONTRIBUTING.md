# Contributing to pi-dynamo-provider

Thanks for your interest in pi-dynamo-provider.

This repository is **not currently accepting external code contributions**. It is a thin compatibility layer between the [Pi agent framework](https://github.com/mariozechner/pi-mono) and [NVIDIA Dynamo](https://github.com/ai-dynamo/dynamo); changes are coordinated tightly with both upstream projects, so we keep the contribution surface closed.

If you would like to:

- **Report a bug** or unexpected behavior — please open a [GitHub issue](https://github.com/NVIDIA-dev/pi-dynamo-provider/issues).
- **Request a feature** or integration improvement — please open a [GitHub issue](https://github.com/NVIDIA-dev/pi-dynamo-provider/issues) describing the use case. We will evaluate whether it belongs here, in Pi, or in Dynamo.
- **Discuss broader Dynamo-related changes** — please use the [ai-dynamo/dynamo](https://github.com/ai-dynamo/dynamo) issue tracker or the [CNCF #ai-dynamo Slack channel](https://communityinviter.com/apps/cloud-native/cncf).

If at some point we open this repository to external contributions, this file will be updated with Developer Certificate of Origin (DCO) sign-off and pull-request guidance.
